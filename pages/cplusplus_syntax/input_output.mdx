## 輸入優化
輸入內容太多，有時候會造成執行時間太慢，cin/cout可以加入這兩行加速。

[WIWIHO的競程筆記-I/O 優化](https://cp.wiwiho.me/io-optimize/)
```cpp copy
#include <iostream>
using namespace std;

int main() {
    ios::sync_with_stdio(0);
    cin.tie(0);
    // your code here
}
```
